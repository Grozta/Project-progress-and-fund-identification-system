#### 项目背景

> 现阶段大型建筑公司内部普遍出现的关于工程进度和资金认定上的尖锐矛盾。项目外包后，产生了由项目部->生产经理->生产队的资金和工程进度认定体系。随着网络时代的发展，在工程队伍中的年轻化的过程中，公正公开的，安全的资金认定考虑越来越成为焦点话题
